
# Introduction 
Python data cleaning and visualization project
The data set used is the "Unicorn Companies" data set.
### This project highlights my data cleaning and visualization skills.

# Tools and Libraries I used :
- Jupyter Notebook
- matplotlib: For Visualization.
- Seaborn: for more complex visualizations.
- pandas: For data manipulation and Cleaning.

 # The Analysis :
 After data cleaning the analysis process answered the following Questions :
 - What are the countries with the most number of unicorn companies?
 - What are the industries with the most number of companies?
 - What time is needed on average to become a unicorn?
 - What are the companies with the biggest return on investment?
 - Who are the investors who invested in more than one unicorn?
